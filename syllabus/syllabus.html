    Course Description.

    An introduction to ordinary differential equations. Topics include methods of solving first-order ordinary differential equations, basic theory of solving n-th order linear equations, solving linear systems of first order equations, and Laplace transforms. Other topics may include numerical methods, infinite series methods, and qualitative analysis of differential equations. Applications will be emphasized. Prerequisite: MATH 222 (minimum grade of C-).

    Section Learning Outcomes

    Course Learning Outcomes.

    • Basic Concepts: students will learn the definition of differential equations and their solutions, and verify solutions to given differential equation or initial value problem.
      ğŸ”—
      ğŸ”—
    • Solving First Order Differential Equations: students will identify first order versus higher order, linear versus nonlinear, and homogeneous versus non-homogeneous differential equations, apply Existence and Uniqueness Theorem for first order linear initial value problems, and solve first order linear differential equations analytically using integrating factors or separation of variables. Students will sketch and interpret slope fields and phase lines to construct solutions qualitatively. Students will apply this to solving applied problems.
      ğŸ”—
      ğŸ”—
    • Solving First Order Linear Systems: students will apply matrix theory and Existence and Uniqueness Theorem to first order linear systems, and rewrite higher order scalar equations as first order systems. Students will verify whether or not given vectors are solutions of a given system or initial value problem. Students will use the Wronskian to verify whether or not a solution set is a fundamental set of solutions, and construct general solutions and solve initial value problems. Students will use the eigenvalue method to solve constant coefficient ho- mogeneous systems: Students will use the method of undetermined coefficients to solve non-homogeneous second order linear equations. Students will apply this to model model unforced mechanical vibration, interpreting solutions graphically and categorizing solutions.
      ğŸ”—
      ğŸ”—
    • Laplace Transforms: Students will compute Laplace transforms using the improper integral definition, use the Heaviside step function in proving and applying the first and second shift theorems, compute Laplace transforms using tables and analytic methods, and solve second order equations using this theory. Students will apply convolutions to the study of the delta function and impulse response. Students demonstrate a conceptual understanding of the meaning of algebraic forms, and recognize differences between algebraic expressions and equations.
